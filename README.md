English
This project is a web application designed to manage a roster of swimmers and their individual swim times. 
The system can automatically organize groups of n swimmers by calculating and balancing their cumulative times for a defined distance. 
The algorithm ensures that all relay teams have comparable total times, enabling the planning of relay events where teams finish with very narrow time gaps. 
This promotes fair competition and motivates participants to give their maximum effort until the end.

Français
Ce projet est une application web conçue pour gérer une liste de nageurs ainsi que leurs temps individuels. 
Le système peut organiser automatiquement des groupes de n nageurs en équilibrant leurs temps cumulés pour une distance définie. 
L’algorithme garantit que toutes les équipes de relais disposent de temps totaux comparables, permettant ainsi de planifier des épreuves de relais où les équipes terminent avec des écarts très serrés. 
Cela favorise une compétition équitable et encourage les participants à donner le meilleur d’eux-mêmes jusqu’à la toute fin.
